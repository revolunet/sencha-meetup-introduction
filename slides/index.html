Sencha Quick Overview

What makes Sencha

so special ?

A Funded startup


  • Raised 29M$ since 2010
  • +100 employees dedicated to the frameworks
  • Focused on entreprise-grade RIA solutions since 2006
  • Support and trainings available
  • Integrated RaphaelJS, JQtouch and TheJIT creators

A long term vision


  • Lead the (web) javascript GUI industry
  • Presence in Desktop + Mobile RIA
  • Server + Client javascript frameworks
  • Global solutions, from prototyping to deployment
  • Connect.js, sencha.io, Architect, GXT, Charts, PhantomJS...

The Sencha paradigm


  • 100% declarative javascript, generates standard HTML5
  • Javascript OOP for reusability
  • Massive, unified widget library
  • Crazy awesome data package
  • Homogenous API & Docs
  • Support for (very) old browsers

Sencha defects


  • Steep learning curve
  • Not open source
  • Hardcore debugging for beginners
  • Slow when not used correctly
  • SEO is not a feature (yet?)

2012: Deep improvements

  • ExtJS 4 / Sencha Touch 2 unified codebase
  • SASS+compass theming
  • MVC for decoupled and reusable components
  • POO with an awesome Class manager
  • Dynamic source loader/builder
  • massive performance improvements

ExtJs Licence


  • GPLv3 licence for open source projects
  • OR
  • Commercial projects needs a per developer licence
  • One licence per major version only
  • 600$/year/developer
